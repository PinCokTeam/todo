# To Do App

## Feature
![image](https://github.com/PinCokTeam/todo/assets/33680799/ddfbd08f-4ccc-4579-98f6-2519f80f4f37)

* Use Dark Mode
* Add Todo Task
* List Todo Task
* Check completed Todo Task
* Delete Todo Task

## Spec

frontend is NextJS, backend is django + DRF

## Install

Use docker compose

```shell
docker compose up -d
```
